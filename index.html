  • • NOM :

    Le nom est simplement le nom que vous voulez donner à votre jeton, comme Bitcoin ou Ethereum.

  • • SYMBOLE :

    Le symbole sera le ticket de votre jeton, tel que BTC ou ETC (normalement les jetons ont un symbole de 3 lettres majuscules).

  • • DECIMALES :

    Décimales est le nombre de décimales que votre jeton peut avoir, ce qui détermine comment elle est divisible. Généralement, les jetons auront 18 décimales, ce qui permet de diviser 1 jeton en trillions de pièces (par exemple, avec 18 décimales, vous pourriez avoir aussi peu que 0,000000000000000001 d'un jeton).

  • • OFFRE TOTALE :

    L'offre totale sera le nombre de jetons disponibles. Offre totale de jetons VALEUR sera le nombre total de jetons à la puissance des décimales. (par exemple, si l'offre totale est de 1000 et décimales est 18 alors donnez 1000000000000000000000 comme valeur.)
